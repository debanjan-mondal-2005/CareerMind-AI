Answering Strategy:
Use the STAR method for behavioral questions:
Situation, Task, Action, Result.

For technical questions:
1. Define the concept.
2. Explain intuition.
3. Give example.
4. Mention project use case.

For project explanation:
1. Problem statement
2. Dataset
3. Preprocessing
4. Model/approach
5. Evaluation
6. Deployment
7. Monitoring
8. Future improvement
